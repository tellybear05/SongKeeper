# SongKeeper

SongKeeper is a C++ console-based song management application developed using Object-Oriented Programming principles.

## Features

* Add songs to the collection
* Search songs by title, artist, album, genre, or year
* Delete songs from the collection
* Save songs to a file
* Load songs from a file
* Case-insensitive searching
* Input validation and exception handling

## Technologies Used

* C++
* Object-Oriented Programming (OOP)
* Dynamic Memory Management
* File Handling
* Exception Handling

## Project Structure

* `main.cpp` – User interface and menu system
* `SongKeeper.h` – Class declarations
* `SongKeeper.cpp` – Class implementations
